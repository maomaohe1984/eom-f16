External Resources

Very Useful

  • processing.org —official Processing website, contains reference, examples, tutorials, exhibition gallery, additional libraries and more.
  • Processing Cheatsheet PDF
  • Daniel Shiffman's Video Lectures —excellent set of video lectures from Dan Shiffman's Intro to Computational Media class, all in Processing
  • OpenProcessing —lots of open source Processing examples. Site is old, dated and often down, but the best single resource. Many examples can't run online, just click the download links.

Books

  • Getting Started with Processing (2nd edition) by Casey Reas and Ben Fry our course text for Processing (2nd edition coming August 2015).
  • Processing: A Programming Handbook for Visual Designers (Second Edition) —a beautiful book on Processing by the language creators Ben Fry and Casey Reas. Contains tutorials on Processing as well inspiring interviews with creators of individual projects.
  • The Nature of Code —read freely in digital form (recommended) which allows viewing of live examples as you read. Excellent reference to browse through at some point, covers particle systems, cellular automata, fractals and other topics based on nature.
  • Learning Processing —a great book to learn Processing at your own pace by Dan Shiffman, larger alternative to our course text.
  • processing.org/books/ —complete listing of Processing related books on the Processing website.

Boston

  • Boston Tech Poetics —an active Boston community of creative coders exists online only as a Google Group, they meet in person for show-and-tell and other events every month or so (formerly Boston Creative Coders).
  • Boston Cyberarts —active Boston organization supporting tech related art, has an excellent Gallery at the Green St. Station of the Orange Line in Jamaica Plain.

Processing Exhibition Websites

  • Beautiful Programming —many examples of programming with source code in Processing
  • 365 Days of Processing —365 visual Processing examples with source code on github (ZIP)
  • Generative Design Code —videos of every example and source code from the book Generative Design: Visualize, Program, and Create with Processing by Hartmut Bohnacker et al. Download all as zip
  • The Recode Project —“An active archive of computer art”, searchable, browse-able all in Processing
  • Neil McLaren —“Sushi-Propelled Idea Monkey”, Neil's Processing sketches and more, almost all have source code.

Creative Coding

  • Creative Applications Network excellent blog on the intersection of creative practice and technology. Good list of creative programming tools
  • Hacking for Artists —decent but dated reference of tools books and people in the creative programming community

Teaching

  • http://shiffman.net/teaching/ —Daniel Shiffman's teaching resources including video lectures. Shiffman is a prominent teaching figure in the Processing community. He teaches at ITP and has published excellent books.
  • Dana Moser has excellent online resources for some of his (and Fred's) MassArt courses in electronics and programming: Electronic Projects for Artists, Electronic Projects for Artists 2 and Digital Projects.

Digital Art Archives

  • ArtBase @Rhizome
  • Archive of Digital Art

About the Course

Elements of Media is a graduate-level studio elective focused upon de- veloping a foundational understanding of the complex synthesis of visual, aural, and temporal information within dynamic media. Through in-class demonstration students will learn the technological and programmatic processes necessary to begin creating visual explorations in image, sound, and interaction. The critical analysis of these explorations, combined with examples and readings, will serve as tools to develop a basic design gram- mar for communication within dynamic media. Prerequisites: none.

Course content, verbiage, examples and philosophies adapted from Brian Lucid's, Alison Kotin's and Nicole Tariverdian's Elements of Media courses.
